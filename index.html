Proposals that look like you mean it

CLI that generates polished client proposals from bullet points. Free markdown output — Pro unlocks PDF export with 6 professional themes.

Get Pro — $19 View on GitHub →

Quick start (free)

npx proposalkit \
  --client "Acme Corp" \
  --project "E-commerce Redesign" \
  --budget "\$8,500" \
  --timeline "6 weeks" \
  --scope "UX audit,Visual design,Frontend development,QA" \
  --about "Two-person design studio, 20+ e-commerce projects."

# → proposal.md

Pro: PDF export

PROPOSALKIT_PRO_KEY=your_key npx proposalkit \
  --client "Acme Corp" \
  --project "E-commerce Redesign" \
  --budget "\$8,500" --timeline "6 weeks" \
  --scope "UX audit,Visual design,Frontend development" \
  --about "Two-person design studio." \
  --pdf --theme modern \
  --output proposal-acme.pdf

# → proposal-acme.pdf (styled PDF, ready to email)
